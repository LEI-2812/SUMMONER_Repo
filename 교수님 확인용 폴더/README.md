## 교수님 확인용 폴더
### 폴더 설명
⏰필요한 내용만 뽑아서 보여드릴 수 있도록 만들어진 폴더입니다. </br>
⏰주로 **구현 기술 관련 문서**가 업로드될 예정입니다. <br/>


### 문서 리스트
추후 업데이트 예정입니다:) </br>

### 사용 기술 관련 업데이트
저희 프로젝트는 '혼합전략'이라는 확률 개념을 사용해 공격을 예측하는 시스템을 개발하고 있습니다. </br>
이에 따른 전투 시스템의 클래스 다이어그램을 이번주 내로 업로드하고, 소스 코드에 대한 설명을 같이 첨부할 예정입니다. </br>
참고 부탁드리겠습니다:D
